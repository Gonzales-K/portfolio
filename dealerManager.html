The Homepage

This is your basic landing page. It features welcome message and image beneath a navigation bar with several links to the various features of the application. The position and size of all elements is responsive to the device viewport.

The Inventory Page

The inventory page pulls the inventory from the database and displays only stock that does not have a sale record. When a sale record is created, a PUT request is sent to the inventory database to update the 'sold' status to 'true'. This update will remove the vehicle from the current inventory display.

The Create Sale

The new sale page contains a form to create a new sale record. The form contains dropdown selections for the current, unsold, stock as well as for selecting the salesperson. Upon submission all fields will be validated. If the entries meet all requirements, a new sale record will be created. The state for each field will then be reset, thereby removing the vehicle from newly created sale from the dropdown options
